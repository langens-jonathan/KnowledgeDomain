KNOWLEDGE DOMAIN

How to use Knowledge Domain

Starting it up

You can use a terminal and go to the location to where the root Knowledge Domain folder is located. And then type 'python KnowledgeDomain/KnowledgeDomainServer.py 4433' where 4433 would be the desired port.

Querying the Knowledge Domain

There are different types of queries. All queries consist of an extended version of the base query

  • Base Queries
  • Object Queries
  • Predicate Queries
  • Predicate Definition Queries
  • Type Queries
